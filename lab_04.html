Preguntas a Resolver | JS

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programación compilado y orientado a objetos, mientras que JavaScript es interpretado y se usa principalmente para desarrollo web. Ambos comparten una sintaxis similar, pero Java es más estricto en tipado y estructura.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

Algunos métodos del objeto Date incluyen getFullYear() para obtener el año, getMonth() para obtener el mes, getDate() para el día, getHours() para las horas y getMinutes() para los minutos.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

Los arreglos tienen métodos como push() para agregar elementos al final, pop() para eliminar el último elemento, shift() para eliminar el primero, unshift() para agregar al principio, y forEach() para recorrer los elementos.

¿Cómo se declara una variable con alcance local dentro de una función?

Se declara con let o const dentro de la función. Esto limita su uso al contexto de esa función y no afecta al resto del programa.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

El uso de variables globales puede causar confusión, errores imprevistos y sobrescritura de valores. Además, dificulta el mantenimiento del código debido a su alcance global.
